# ♟️ ChessMind

**ChessMind** — это современная шахматная платформа, разработанная на фреймворке Angular. Проект сочетает в себе классическую игровую механику, продуманный интерфейс и эвристический алгоритм принятия решений.

## 🚀 Основные возможности

* **Интеллектуальный ИИ**: Реализован собственный алгоритм на базе эвристического анализа позиции. Бот оценивает доступные ходы, используя весовые коэффициенты фигур и позиционные преимущества.
* **Три уровня сложности**:
    * **Easy**: Случайный выбор ходов для новичков.
    * **Medium**: Базовый анализ материального веса фигур.
    * **Hard**: Продвинутая оценка с учетом контроля центрального поля и стратегического расположения фигур.
* **Режим PvP (Person vs Person)**: Локальный режим игры для двух человек с полной валидацией шахматных правил.
* **Интеграция с Supabase**: 
    * Полноценная система авторизации (Sign In / Sign Up).
    * Облачная база данных для хранения профилей пользователей.
* **Премиальный UI/UX**: 
    * Эстетичная тёмная тема оформления.
    * Уникальный брендинг и кастомная иконка приложения (favicon).
    * Адаптивный дизайн для комфортной игры на разных устройствах.

## 🛠 Технологический стек

* **Frontend**: Angular 17+ (компонентная архитектура, TypeScript).
* **Backend & Auth**: Supabase (PostgreSQL).
* **Логика**: Интеграция с chess.js для валидации ходов и реализации эвристических алгоритмов.
