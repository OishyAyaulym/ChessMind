# ♟️ ChessMind

**ChessMind** — это высокотехнологичная шахматная платформа, разработанная на фреймворке Angular. Проект сочетает в себе классическую игровую механику, современный интерфейс и алгоритмический искусственный интеллект.

## 🚀 Основные возможности

* **Интеллектуальный ИИ**: Реализован собственный шахматный движок на базе алгоритма **Minimax** с оптимизацией **Alpha-Beta pruning**. Бот анализирует дерево ходов и выбирает позицию с максимальным весом фигур.
* **Режимы игры**: 
    * **PvE**: Тренировки против ИИ с динамическим расчетом ходов.
    * **PvP**: Локальный режим для игры двух человек с полной валидацией шахматных правил.
* **Система авторизации**: Интеграция с **Supabase Auth** для регистрации и входа пользователей.
* **Облачная база данных**: Хранение профилей пользователей и статистики в реальном времени через Supabase.
* **Премиальный UI/UX**: 
    * Тёмная тема оформления.
    * Кастомный брендинг и уникальный логотип (favicon).
    * Адаптивная верстка для корректного отображения в браузере.

## 🛠 Технологический стек

* **Frontend**: Angular 17+ (компонентная архитектура, TypeScript).
* **Backend & Auth**: Supabase (PostgreSQL).
* **Алгоритмы**: Minimax, Alpha-Beta pruning, Piece-Square tables для оценки позиции.
* **Стилизация**: CSS3 с использованием переменных для темизации.

## 📁 Структура проекта

* `src/app` — основная логика компонентов (игра, авторизация, навигация).
* `src/assets` — графические ресурсы и иконки.
* `environments` — конфигурация для подключения к API Supabase.
